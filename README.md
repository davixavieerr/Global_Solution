# Space Predictive Analytics

Sistema de telemetria e monitoramento espacial desenvolvido para a Global Solution 2026.1 - FIAP.

## Descrição
O **Space Predictive Analytics** é uma solução *cross-platform* (Web/Mobile) para monitoramento de sistemas críticos em missões espaciais. O app fornece dashboards em tempo real para controle de temperatura, energia e qualidade de sinal, permitindo a gestão de alertas preventivos.

## Funcionalidades
- **Telemetria ao Vivo:** Dashboards dinâmicos com dados simulados.
- **Configuração de Missão:** Cadastro de parâmetros críticos via formulário.
- **Persistência de Dados:** Configurações salvas localmente via AsyncStorage.
- **Tema Dinâmico:** Alternância entre temas claro e escuro.
- **Gestão de Alertas:** Central de notificações para falhas de sistema.

## Tecnologias Utilizadas
- React Native
- Expo Router
- TypeScript
- Context API (Estado Global)
- AsyncStorage (Persistência)

## Como rodar o projeto
1. Clone este repositório.
2. Instale as dependências: `npm install --legacy-peer-deps`
3. Execute o projeto: `npx expo start --web`
