# 🚀 Space Predictive Analytics
### Global Solution 2026.1 - Cross-Platform Application Development | FIAP

## 1. Descrição
O **Space Predictive Analytics** é uma solução *cross-platform* (Web/Mobile) desenvolvida para monitoramento de telemetria espacial em tempo real. O aplicativo soluciona a necessidade de um painel de controle portátil e eficiente para monitorar sistemas críticos, como temperatura interna, carga de bateria e estabilidade orbital. Nosso diferencial reside na integração de persistência de dados local com uma interface responsiva, permitindo que operadores de missão gerenciem limiares de segurança e recebam alertas críticos de forma imediata.

## 3. Telas do Aplicativo

### Home - Dashboard Principal
Visão geral dos indicadores da missão: energia, temperatura, sinal e estabilidade orbital.

### Dashboard de Sensores
Gráfico de linha com leituras dos sensores em tempo real simulado.

### Dashboard de Energia
Indicadores de carga dos painéis solares e consumo dos sistemas.

### Dashboard de Comunicação
Status do link de telemetria, latência e qualidade do sinal.

### Alertas
Lista de alertas ativos gerados automaticamente com nível de criticidade.

### Configurações / Formulário
Formulário de configuração dos limiares de alerta com validação.

## 4. Funcionalidades
- [x] Dashboard com indicadores em tempo real (simulado)
